## Installation

**NPM**

```bash
npm install sysacss
```

**CDN**

```html
<!-- Unpkg -->
<link rel="stylesheet" href="https://unpkg.com/sysacss" />
```

```html
<!-- jsDelivr -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/shoaiyb/sysacss@master/sysa.css" />
```


**Download**

[`sysacss@master`](https://github.com/shoaiyb/sysacss/archive/refs/heads/master.zip)


## Features
- [x] Normalized by default.
- [x] No preprocessor, just plug-n-play.
- [x] No conflicts with other used frameworks, all classes start with the `ss-` prefix.
- [x] It's light, less than 40Kb raw.

## Browser support

- [x] Chrome
- [x] Edge
- [x] Firefox ESR+
- [x] Internet Explorer 10+
- [x] Opera
- [x] Safari 8+
