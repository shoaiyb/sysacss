## Features

- Normalized by default.
- No preprocessor, just plug-n-play.
- No conflicts with other used frameworks, all classes start with the `ss-` prefix.
- It's modular, import what you need.

## Browsers support

- Edge
- Firefox ESR+
- Internet Explorer 10+
- Opera
- Safari 8+
